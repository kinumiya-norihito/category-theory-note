圏

圏 𝒜 は以下のように構成される。

  • ob ⁡ ( 𝒜 ) が存在し、その要素を対象(object)と呼ぶ。

  • A , B ∈ ob ⁡ ( 𝒜 ) に対して、 𝒜 ⁡ ( A , B ) ∈ mor ⁡ ( 𝒜 ) が存在し、 mor ⁡ ( 𝒜 ) の要素を射(mor-phisms)と呼ぶ。

  • A , B , C ∈ ob ⁡ ( 𝒜 ) に対して、 𝒜 ⁡ ( A , B ) , 𝒜 ⁡ ( B , C ) ∈ mor ⁡ ( 𝒜 ) がある時、 𝒜 ⁡ ( B , C ) × 𝒜 ⁡ ( A , B ) → 𝒜 ⁡ ( A , C ) ; ( g , f ) ↦ g ∘ f となるような ∘ が存在し、これを合成(composition)と呼ぶ。

  • A ∈ ob ⁡ ( 𝒜 ) に対して、 𝒜 ⁡ ( A , A ) が存在し、このような射を恒等射(identity)と呼び、 id A と表す。

また、以下の公理を満たす。

  • 結合律

    f ∈ 𝒜 ⁡ ( A , B ) , g ∈ 𝒜 ⁡ ( B , C ) , h ∈ 𝒜 ⁡ ( C , D ) に対して、 ( h ∘ g ) ∘ f = h ∘ ( g ∘ f ) が成り立つ。

  • 恒等性

    f ∈ 𝒜 ⁡ ( A , B ) に対して、 f ∘ id A = f = id B ∘ f が成り立つ。

また、表記の簡略化のために、以下は表記と意味の対応である。

  • A ∈ 𝒜 は A ∈ ob ⁡ ( 𝒜 ) を意味する。

  • f : A → B または A → f B は f ∈ 𝒜 ⁡ ( A , B ) を意味する。

  • g ⁡ f は g ∘ f を意味する。
